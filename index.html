What tools do I use?

Front end tools

Find out more!

Vue.js is a javascript MVC (Model View Controller) library/framework, here's an interesting article that mentions the prominent UI frameworks, which includes React.js, Angular.js and of course, Vue.js.

Vue.js allows me to do many great things like control the data displayed on a website, display components, apply javascript hooks, manage events, add animations with third party support such as animate.css, etc. As you can see, it can do many things that jQuery does, however, Vue.js can do better and more elegantly, if you want to learn more just check out their website, I'm sure you'll love it.

Find out more!

Bulma is a wonderful CSS framework, I'm actually using it right now in this site! As you can see, Bulma allows me to display simple elements with modern styling without many problems. I have found that Bulma works very nicely with javascript MVC libraries like Vue thanks to Bulma's many class uses such as the navbar burger, dynamic background colors, toggling classes and many more things.

Find out more!

Bootstrap is the first CSS framework I learned, it's a very powerful tool for styling web pages but it comes with some inconveniences such as already having high priority predefined style for generic tags such as <h1-6>, <ol>, <blockquote>, <p>, etc. This framework is what I call CSS with batteries included, which is why I prefer Bulma.

Fun fact: I decided to move to Bulma after Bootstrap released its 4th version, I noticed that their new version borrowed some things from bulma and over complicated itself, their naming conventions didn't appeal to me and after checking out Bulma, memorizing class names and applying them seemed more intuitive.
