The technologies I used for this project were so many at the time, I first initialized this project when I started learning Vue.js, now that I think about it, finishing this project without it would've been pretty hard for the beginner me.

This was actually my first website that contained various views so mind the rough look (what with the plain color usage, lack of images and whatnot), I named the views as "Home", "Cooking", "Video Games" and "Programming", every one of them was called after things I like very much, except "Home" obviously, that's just the root of the site. The "Cooking" and "Programming" section take their looks from mostly blogs, specially the "Programming" one, it's designed to display useful information in a readable format accesible to all browsers and viewports, the cooking section is designed to display many awesome recipes in a creative list where the items resemble hamburgers while still keeping harmony with the color profile of the section itself.

Probabilites Generator

A dynamic way to input combinatory aleatory experiments data and get the result of the experiment through a special algorithm I made

This is something I made when I was learning about probability at my university, it's basically a simulation of the "balls in a box" scenario, the user can add as many balls that he wants inside a box, and can calculate the probability of a certain event.

This project uses Vue.js and Plotly.js which is used for graphing data in a variety of ways. For the style I just went with plain Bulma CSS.
